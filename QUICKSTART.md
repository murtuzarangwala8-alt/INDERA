# INDÉRA — Quick Start Guide

## 🚀 Get Started in 3 Steps

### 1. Install Dependencies
```bash
npm install
```

### 2. Start Development Server
```bash
npm run dev
```

The site will open at **http://localhost:5173**

### 3. Build for Production
```bash
npm run build
npm run preview
```

---

## 📱 What You'll See

### Homepage Sections (in order):
1. **Hero** — 3D floating jewelry with "Indian Heritage. European Elegance."
2. **Categories** — 5 collection cards (Minimal Jhumkas, Pearl Fusion, etc.)
3. **Featured Products** — 4 bestseller/new product cards
4. **Trust Badges** — 4 brand pillars (Artisan Crafted, Indo-European Design, etc.)
5. **Brand Story** — Image + text about Indian craftsmanship + European minimalism
6. **Artisan** — 4 city cards (Jaipur, Delhi, Mumbai, Surat)
7. **Lookbook** — Editorial grid layout
8. **Testimonials** — 4 customer reviews
9. **Newsletter** — Email signup for "INDÉRA private list"

### Other Pages:
- **/products** — Full catalog with category filter, search, sort
- **/product/:id** — Individual product detail page
- **/cart** — Shopping bag with checkout CTA
- **/wishlist** — Saved items
- **/about** — Full brand story + artisan section + values
- **/contact** — Contact form (existing)

---

## 🎨 Design Features

- **Colors:** Ivory background, matte gold accents, obsidian text
- **Typography:** Cormorant Garamond (serif) + DM Sans (sans-serif)
- **Animations:** Framer Motion scroll-triggered animations
- **3D Effects:** Product cards with hover transforms, glassmorphism
- **Responsive:** Mobile-first, fully responsive

---

## 🛠️ Tech Stack

- React 18 + TypeScript
- Tailwind CSS (custom luxury theme)
- Framer Motion (animations)
- React Router v6
- Vite (build tool)
